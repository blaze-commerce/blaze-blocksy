## 2. Product Slider H3 Titles

**Problem:** Product titles under "Favourite Buys" were `<h2>`, same level as section headings. Flat hierarchy.

**Fix:** Temporary `theme_mod_woo_card_layout` filter in the product slider shortcode sets `heading_tag: "h3"` only during the slider's WC loop.

**Scope:** Only affects `[bc_product_slider]` shortcode. Category titles, New Arrivals, shop/archive pages all stay `<h2>`.

**Files:**
- `inc/product-slider.php` — added `bc_slider_h3_product_titles()` filter
- `docs/patterns/product-slider-h3.md` — NEW pattern doc

**Heading hierarchy after fix:**

## 4. Header Search Bar — Full-Width FiboSearch

**Problem:** FiboSearch search bar was cramped in the header. Blocksy's CSS Grid (`grid-template-columns: 1fr var(--middle-column-width, auto) 1fr`) constrained the middle column to content width.

**Fix:** Broke the 3-column CSS Grid into a single flex row using `display: contents` on column wrappers. Search bar gets `flex: 1` to fill available space.

### Architecture

```
blocksy-child/
├── assets/css/components/
│   ├── fibosearch.css         ← NEW: all FiboSearch plugin styling
│   │   - Desktop: display:contents flattens 3 columns → 1 flex row
│   │   - Search: flex:1, min-width:701px fills available space
│   │   - Dividers: explicit 1px × 20px after column flattening
│   │   - Mobile: full-width search in bottom row
│   │   - All rules scoped to [data-device="desktop"] or [data-device="mobile"]
│   │
│   └── header.css             ← REWRITTEN: Blocksy header tweaks only
│       - Announcement bar centering
│       - Desktop bottom row hide (scoped to [data-device="desktop"])
│       - Contacts nowrap
│       - NO FiboSearch rules (moved to fibosearch.css)
│
├── clients/byronbay/
│   └── byronbay.css           ← MODIFIED: removed FiboSearch section
│
├── inc/
│   └── enqueue.php            ← MODIFIED: conditional fibosearch.css loading
│                                 only when plugin is active
│
└── docs/patterns/
    ├── visual-heading-override.md  ← NEW
    └── product-slider-h3.md        ← NEW
```

### Key CSS Technique

```css
/* Flatten Blocksy's 3-column grid → single flex row */
#header [data-device="desktop"] [data-row="middle"] > .ct-container {
    display: flex !important;
    align-items: center;
    gap: 16px;
}

#header [data-device="desktop"] [data-row="middle"] > .ct-container > [data-column] {
    display: contents; /* children become direct flex items */
}

/* Search fills all remaining space */
#header [data-device="desktop"] [data-row="middle"] [data-id="search"] {
    flex: 1 1 0% !important;
    min-width: 701px;
}
```
